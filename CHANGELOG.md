# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-21

### Added

- Initial release of Loom Framework
- **Loom Core Plugin**
  - Layout components: Column, Row, Box, Spacer, Divider
  - Basic components: Text, Button, IconButton, Image
  - Container components: Card, Surface, Dialog, Scaffold
  - Feedback components: Alert, Badge, Progress, Chip, Tooltip
  - Input components: TextField, TextArea, Checkbox, Switch, Select, Slider
  - Modifier API with fluent chainable styling
  - Design tokens: Colors, Spacing, Shapes
  - TextStyle and ButtonStyle enums
  - Head component for meta management

- **Loom Theme Design Plugin**
  - WordPress admin interface for design tokens
  - Color customization
  - Typography settings
  - Spacing configuration

- **Loom Icons Plugin**
  - SVG icon system
  - Multiple icon pack support
  - IDE autocomplete support

- **Loom Notifications Plugin**
  - Toast notification system
  - PHP and JavaScript APIs
  - Multiple notification types

- **Loom Theme**
  - Welcome page demonstrating all components
  - Minimal theme setup
  - Full Loom Core integration

### Documentation

- Complete API documentation
- Claude Code integration guide
- Installation instructions
- Component examples
